Meta Campus: Project REC

An AI-powered immersive virtual campus experience designed for interactive exploration.

Project Overview

← Back to Website

Meta Campus: Project REC is an immersive Virtual Reality–based digital campus developed to transform how users explore, understand, and interact with institutional environments. The project recreates a real-world campus as a high-fidelity 3D virtual space, allowing users to navigate buildings, infrastructure, and facilities in an intuitive and engaging manner.

Unlike traditional virtual tours, Project REC integrates AI-driven narration and interaction, enabling users to communicate naturally with a virtual guide. This approach enhances accessibility, engagement, and information retention.

Project Gallery

Key Features

  • Photorealistic 3D Campus Environment: Precisely modeled buildings, interiors, pathways, and landmarks.
  • AI-Driven Virtual Guide: An intelligent character powered by conversational AI.
  • Natural Interaction System: Supports voice-based and interactive triggers.
  • Optimized VR Performance: Asset optimization for smooth frame rates.

Technology Stack

  • Engine: Unity 3D
  • Language: C#
  • SDKs: Oculus SDK, UnityXR
  • Modeling: Blender
